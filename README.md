# Сайт врача-косметолога — Юлия Васильева

Лендинг для врача-косметолога и дерматолога Васильевой Юлии Андреевны (Иркутск).

## Стек

- **Vite + React + TypeScript** — сборка и UI
- **Tailwind CSS v4** — стилизация
- **Framer Motion** — анимации появления секций
- **Swiper** — карусель отзывов
- **Lucide React** — иконки
- Шрифты: Cormorant Garamond (заголовки) + Montserrat (текст)

## Запуск

```bash
npm install      # установка зависимостей
npm run dev      # дев-сервер (http://localhost:5173)
npm run build    # production-сборка в dist/
npm run preview  # предпросмотр собранной версии
```

## Структура

```
src/
  components/   секции страницы (Navbar, Hero, About, Services, Education, Reviews, Contact, Footer)
  data.ts       контент: услуги, образование, отзывы, статистика
  index.css     тема Tailwind (цвета, шрифты, утилиты)
  App.tsx       сборка секций
```
